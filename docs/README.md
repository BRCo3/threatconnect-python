# threatconnect-app-docs

Source code for apps documentation.

1. Install mkdocs:	<br/>
`http://www.mkdocs.org/#installation`

2. `git clone https://github.com/ThreatConnect-Inc/threatconnect-app-docs.git`

3. `cd threatconnect-app-docs`

4. Run: <br/>
`mkdocs serve`

5. Open browser: 	<br/>
`http://127.0.0.1:8000/`
